# Module 0 · Python Refresh

Este módulo repasa conceptos base de Python para arrancar el curso con un
lenguaje común sobre tipos de datos, estructuras y operaciones esenciales.

## Lecciones

- `01_data_types_and_variables.py`: tipos numéricos, booleanos, strings,
  variables, conversiones y truthy/falsy.
- `02_core_data_structures.py`: listas, tuplas, conjuntos, diccionarios y
  estructuras lineales simples como stack y queue.
