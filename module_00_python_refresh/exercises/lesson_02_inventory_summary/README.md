# Exercise · Inventory Summary

Implementa una función que reciba una lista de productos y construya un resumen
de inventario.

## Objetivos

- Recorrer listas de diccionarios.
- Acumular valores numéricos.
- Usar conjuntos para detectar categorías únicas.

## Resultado esperado

La función debe retornar un diccionario con:

- `total_products`
- `total_stock`
- `categories`
